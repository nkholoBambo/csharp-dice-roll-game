# csharp-dice-roll-game
C# application that randomly  rolls dice and displays a rewards message for each category

The program simulates rolling three dice simultaneously and calculates the total score. Based on the total, the player receives different rewards and custom messages.

##Features
-Simulates rolling 3 dice
-Random number generation
-Reward system based on total score
-Custom message for each outcome

##How it works
-Rolls 3 dice (randomly from 1 to 6)
-The values are displayed to the user
-Total score is calculated
-Reward is given based on score range

##Example
Dice roll: 1 + 1 + 1
You win a kitten
